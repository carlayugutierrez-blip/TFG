# TFG - The Gardenhart - Carla Yu Gutiérrez, Patricia Aragón
Hola! Somos Patricia Aragón y Carla Yu Gutiérrez, estudiantes del CITM (UPC). Para TFG, estamos desarrollando un escenario navegable como propuesta conceptual para un hipotético videojuego. Actualmente, el proyecto se encuentra en sus fases iniciales de prototipado.

La siguiente actividad consiste en navegar libremente por el escenario, observando el entorno y prestando atención a elementos destacables. Una vez finalizada la exploración, les agradeceríamos que respondan el cuestionario adjunto. La obtención de respuestas precisas y detalladas nos permitirá mejorar y contribuir positivamente al desarrollo del proyecto.

A continuación, puedes encontrar el test en el apartado "releases".

Muchas gracias por participar!
